# UNDP Cabo Verde #

## Requirements ##

  **Ruby version:** mri 2.2.2

## SETUP ##

Just execute the script file in bin/setup

  Depends on UNDP Cabo Verde [repository](https://github.com/Vizzuality/undp_cabo_verde)


### Install gems: ###

    bundle install

### Run application: ###

    foreman start

## TEST ##

  Run rspec: 
  ```ruby
    bin/rspec
  ```
  Run teaspoon: 
  ```ruby  
    rake teaspoon
  ```
  Run cucumber: 
  ```ruby  
    rake cucumber
  ```
  Run all: 
  ```ruby
    rake
  ```
